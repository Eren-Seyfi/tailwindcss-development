# tailwindcss-development skill

This skill teaches agents how to write and maintain Tailwind CSS in **Vite + React** projects.

## When it activates

- User mentions "tailwind" in any form
- Building responsive grid or flex/grid layouts
- Styling UI components (cards, tables, navbars, forms, badges)
- Adding dark mode variants
- Fixing spacing or typography
- Tailwind v3/v4 setup or migration
- React + Vite + Tailwind configuration

## What it covers

| Topic | Details |
|---|---|
| Setup | Tailwind v4 (CSS-first) and v3 (PostCSS) |
| Dark mode | OS preference and manual class toggle |
| Responsive | Mobile-first breakpoints, custom tokens |
| Layouts | Flex, grid, app shell, fixed topbar |
| Components | `clsx`, `tailwind-merge`, `cva` patterns |
| Animations | Built-in transitions, custom keyframes |
| Pitfalls | v3→v4 migration traps, dynamic class names |

## Skips

Backend logic, database queries, API routes, vanilla CSS unrelated to Tailwind.
